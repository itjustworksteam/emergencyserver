Emergency Server API

GET /api/all

Return a JSON array with Country objects.

Country

Field Type Description
name string The name of the country
code string The code of the country
police string The police number of the country
medical string The medical number of the country
fire string The fire fighters number of the country

GET /api/:country

Return a JSON object with a Country object if found else it will return this JSON: {"error":"No country found with the given 2-letter country code hj"}

GET /api/:latitude/:longitude

Return a JSON object with a Country object if found else it will return this JSON: {"error":"Sorry cannot get your location. Retry!"}
